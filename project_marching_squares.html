Marching Squares

About

Marching squares is a computer graphics algorithm that generates contours for a two-dimensional scalar field.
For more information read the wikipedia entry.

In this project I created three (slightly) different implementations. The general idea was that I should compare the three versions and chose one based on personal preference and some sort of metrics.
The three versions are:

  1. OOP and the standard library
  2. OOP but not the standard library
  3. neither OOP nor the standard library

My goal with this project was:

  • - Re-learn DirectX (it was quite some time ago since I last looked at it)
  • - Try out various coding styles and libraries (e.g. using the standard library vs not using the standard library)
  • - Create a small, but cool, project that could be finished in a reasonable time frame
  • - Get some experience in optimizing algorithms and code
  • - Create a (somewhat) efficient implementation of the marching squares algorithm

The source code is hosted on GitHub.

Notes

The development was documented in text-files, keeping it simple and to the point. The text-files were then converted into html by a simple c++ program.
